# 📝 To-Do List Web App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Clear all tasks
- Data stored using localStorage (persists after refresh)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation, LocalStorage)

## 📂 Project Structure
index.html ,
styles.css ,
script.js
